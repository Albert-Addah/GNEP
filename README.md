# NEP – National Education Platform 🇬🇭

A unified learning and examination system designed to eliminate educational inequality in Ghana.

## Features
- Offline-first learning via school servers
- AI Companion for guided learning
- Structured lectures and notes
- Smart quiz system
- Secure exam hall for national exams
- Subscription-based access

## Modules
- Mobile App
- Backend Services
- Admin Dashboard
- School Server System

## Vision
To provide every Ghanaian student access to high-quality education regardless of location.
